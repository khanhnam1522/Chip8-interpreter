How to use

In order to use the Chip8 sprite editor, click on any pixel in the 8x15 grid where they want the sprite drawn in. Users can reset a pixel by clicking in that same position again, or by clicking the clear button below. The following numbers printed at the bottom of the page represents the current sprite in hexadecimal. The hexadecimal values can be copy and pasted to whatever Chip8 assembler/editor the user is programming in for their own use.
